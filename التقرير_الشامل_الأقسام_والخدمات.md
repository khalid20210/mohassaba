## 🏢 الأقسام والخدمات حسب نوع النشاط

### 📦 1. التجزئة العامة (RETAIL) — البقالات، السوبر ماركت، محلات عامة

**الخدمات المتخصصة:**

| الخدمة | الميزات | المسار |
|--------|--------|--------|
| **نقاط البيع (POS)** | باركود + سلة شراء + دفع فوري | `/pos` |
| **المخزون** | تتبع الكميات، تنبيهات الأقل | `/inventory` |
| **المشتريات** | إضافة مشتريات من الموردين | `/purchases` |
| **إدارة الباركود** | إنشاء + طباعة + تحديث | `/barcode` |
| **الفواتير** | فواتير بيع + شراء | `/invoices` |

**الـ pos_mode:** `standard`
- البحث الأساسي: **باركود**
- لا يوجد: صور، طاولات، صلاحية

### 🍽️ 2. المطاعم والكافيهات (RESTAURANT)

| **الوصفات** | تعريف الأصناف والمكونات | `/recipes` |

**الـ pos_mode:** `restaurant`
- البحث الأساسي: **اسم المنتج**

## 🚀 ملخص الميزات الرئيسية

✅ **نظام متشكّل** — كل قطاع له واجهة مخصصة  
✅ **OCR ذكي** — استيراد الفواتير من صور ومستندات PDF  
✅ **مخزون حي** — تتبع لحظي للكميات والصلاحية  
✅ **نقطة بيع مرنة** — باركود + صور + طاولات + أسعار ديناميكية  
✅ **تقارير شاملة** — مالية وضريبية (VAT + ZATCA)  
✅ **موارد بشرية** — رواتب + عمولات + صلاحيات  
✅ **API-First** — لربط المنصات الخارجية  
✅ **قمرة قيادة** — لوحة مالك المنشأة مع سجل نشاط كامل
