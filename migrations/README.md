# مجلد migrations/ — نظام تهجير قاعدة البيانات
## كيف يعمل؟
- كل ملف `.sql` يُطبَّق **مرة واحدة فقط** ثم يُسجَّل في جدول `_schema_migrations`
- الترتيب رقمي تصاعدي: `001_*.sql` → `002_*.sql` → ...
- **لا يُحذف بيانات** — كل التغييرات بـ `CREATE IF NOT EXISTS` أو `ALTER TABLE ADD COLUMN`

## إضافة migration جديدة
1. أنشئ ملف `NNN_اسم_وصفي.sql` في هذا المجلد
2. اكتب فقط الجُمَل الجديدة (لا تعيد كتابة الجداول القديمة)
3. شغّل التطبيق — سيتم تطبيقه تلقائياً

## تفعيل ملف `.pending`
الملفات بلاحقة `.pending` لا تُطبَّق — تُستخدم كمسودة جاهزة.
لتفعيل ملف: **أزل `.pending` من اسمه**.
